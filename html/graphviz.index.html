Index of sample SVG files created by GraphViz2.

All scripts, input files and output files required to generate these examples are shipped in the GraphViz2 distro.

1: Demonstrates the transitive 6-net, also known as Heawood's graph.
2: Demonstrates empty strings for node names and labels.
3: Demonstrates a cluster - with a bug. See the TODO in the POD.
4: Demonstrates graphing a database schema.
5: Demonstrates graphing a Perl data structure.
6: Demonstrates using XML::Bare to parse HTML.
7: Demonstrates graphing a Parse::RecDescent-style grammar.
8: Demonstrates graphing a Perl regular expression.
9: Demonstrates using XML::Bare to parse XML.
10: Demonstrates using XML::Tiny to parse XML.
11: Demonstrates graphing a byacc-style grammar.
12: Demonstrates graphing a yapp-style grammar.
13: Demonstrates (1) newlines and double-quotes in node names and labels, (2) justification - with a Graphviz bug.
14: Demonstrates a subgraph (with a frame because the subgroup is called cluster_*).
15: Demonstrates a subsubgraph.
16: Demonstrates a trivial 3-node graph, with colors.
